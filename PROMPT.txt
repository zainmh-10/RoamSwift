## Key Architecture

**Technology Stack**: Next.js 16 + React 19 + TypeScript 5.9 + Tailwind CSS v4
- Uses **Turbopack** for fast development builds
- **Geist** fonts (Sans & Mono) via next/font
- Dark mode support with `dark:` Tailwind classes
- Custom animations (blob effects) in globals.css

**App Router Structure**:
- `layout.tsx` - Root layout with fonts, metadata, and global styles
- `page.tsx` - Home page rendering Hero component
- `globals.css` - Tailwind v4 setup with custom theme and animations

**Documentation**:
- `docs/CODEBASE_MAP.md` - Comprehensive project overview, architecture, and navigation
- `docs/convex/` - Complete Convex database integration guides
- `docs/clerk/` - Clerk authentication setup and patterns
- `for-agent/` - AI agent skills and configuration (excluded from git)
